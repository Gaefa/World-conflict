# Conflict.Game

Геополитическая MMO-стратегия в реальном времени на 3D-глобусе. Управляй реальной страной — экономикой, армией, дипломатией, разведкой и технологиями. Каждое решение влияет на мир, где играют другие люди.

## Суть игры

Ты выбираешь одну из 30 реальных стран (США, Россия, Китай, Япония, Бразилия и др.) и управляешь ей в реальном времени. У каждой страны свои стартовые условия — GDP, армия, ресурсы, технологии, население — на основе реальных данных.

Игра идёт тиками (1 тик = 1 месяц игрового времени). Каждый тик обновляется экономика, двигаются армии, меняются отношения. Ты принимаешь решения — сервер считает последствия.

## Что можно делать

### Экономика и ресурсы
- Добыча и переработка 25+ видов ресурсов (нефть, газ, металлы, редкоземельные, продовольствие)
- Цепочки переработки: руда + уголь → сталь, редкоземельные + медь → электроника → полупроводники
- Глобальный рынок с ценами на основе спроса/предложения
- Торговые сделки в стиле Civilization — предлагай ресурсы, торгуйся, ставь скидки
- Контрабанда для обхода санкций (с риском обнаружения)
- Манипуляция ценами (OPEC-стиль: сократи добычу → цена растёт)
- Стратегические запасы на случай блокады

### Военное дело
- Армия, флот, ВВС — перемещение юнитов по глобусу
- Объявление войны, вторжение, морская блокада
- Ядерный удар, ракетный удар, кибератаки
- Размещение военных баз на территории союзников
- Тайные операции: переворот, саботаж, поддержка повстанцев

### Дипломатия
- Альянсы, пакты о ненападении, торговые соглашения
- Санкции — блокируют торговые потоки, рвут цепочки поставок
- Голосования в ООН, ультиматумы, мирные переговоры
- Гуманитарная помощь, экономическая помощь союзникам

### Разведка (Fog of War)
- Каждый игрок видит только то, что разведал
- Шпионские операции: разведка экономики, военных, политики, кибер
- Контрразведка для защиты от шпионов
- Дезинформация — подбрасывай ложные данные противнику
- Досье на страны — собирай информацию тик за тиком

### Технологии
- 36 технологий в 6 ветках: военные, экономические, кибер, космос, биотех, инфраструктура
- Дерево зависимостей — нужно исследовать базовые технологии перед продвинутыми
- Технологии открывают новые действия, бонусы к атаке/защите, торговле, стабильности

### Внутренняя политика
- Стабильность и одобрение населения
- Законы: военное положение, мобилизация, налоговая реформа
- Пропаганда для поднятия одобрения

## Условия победы

| Победа | Как достичь |
|--------|-------------|
| Доминирование | Военный контроль 50%+ мировых victory points |
| Экономическая гегемония | Крупнейшая экономика + контроль 60% торговых маршрутов |
| Дипломатическая | 80%+ стран в твоём альянсе |
| Технологическая | Все технологии финального тира |
| Выживание | Наивысший Index of Power к концу сессии |

**Index of Power** = GDP (25%) + Военная мощь (25%) + Дипломатическое влияние (20%) + Технологии (15%) + Стабильность (15%)

## Стек

| Компонент | Технология |
|-----------|-----------|
| Монорепо | Turborepo |
| Сервер | Fastify + WebSocket (ws) |
| БД | PostgreSQL + Drizzle ORM |
| Веб-клиент | Next.js + Tailwind + Zustand |
| 3D-глобус | react-globe.gl (Three.js) |
| Десктоп | Electron |
| Игровая логика | TypeScript (shared между клиентом и сервером) |

## Структура проекта

```
apps/
  server/     — Fastify API + WebSocket + Game Loop
  web/        — Next.js веб-клиент с 3D-глобусом
  desktop/    — Electron обёртка (DMG/EXE)
packages/
  shared-types/  — TypeScript типы (страны, действия, события, дипломатия, технологии)
  game-logic/    — Чистая игровая логика (экономика, ресурсы, бои, стабильность)
  utils/         — Геоутилиты, форматирование
```

## Запуск (разработка)

```bash
npm install
npx turbo dev        # сервер (порт 3002) + веб (порт 3000)
```

Открой http://localhost:3000 — создай сессию, выбери страну, начни игру.

## Сборка десктоп-приложения

```bash
cd apps/desktop
npm install
npm run dist:mac     # DMG для macOS
npm run dist:win     # EXE для Windows
```
